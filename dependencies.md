# Project Dependencies

## Build Tools

* **gcc-arm-none-eabi**: Cross-compiler for Cortex-M7/M4
* **binutils-arm-none-eabi**: Linker and binary manipulation tools
* **libnewlib-arm-none-eabi**: Standard C library for embedded systems
* **build-essential**: GNU Make and local build utilities
* ccache: Compiler cache to speed up rebuilds

## Hardware & Debugging

* **openocd**: Interface for flashing via ST-LINK
* **gdb-multiarch**: Debugger for ARM cores

## Optional Tools

* bear: generates compilation database for code analysis tools
